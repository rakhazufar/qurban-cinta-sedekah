# Cinta Sedekah - Waktu Indonesia Berkurban (WIB)

Landing page responsif untuk program **Waktu Indonesia Berkurban (WIB)** oleh Cinta Sedekah.

## ✨ Fitur Utama
- **Katalog Hewan**: Daftar harga dan bobot domba/sapi.
- **Carousel Dokumentasi**: Galeri kegiatan dan fasilitas peternakan.
- **Navigasi Pintar**: Highlight menu otomatis saat scroll (Scroll-spy).
- **Responsif**: Optimal di perangkat mobile dan desktop.
- **Integrasi WhatsApp**: Tombol bantuan langsung ke admin.

## 🛠️ Teknologi
- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **Framework**: Bootstrap 5, Font Awesome 6, Google Fonts (Poppins)

## 📁 Struktur Folder
```text
qurban/
├── index.html          # Halaman utama
├── assets/
│   ├── css/style.css   # Styling kustom
│   ├── js/script.js    # Logika eksternal (Carousel & Scroll)
│   └── images/         # Aset gambar
└── README.md           # Dokumentasi
```

## 🚀 Cara Menjalankan
Cukup buka file `index.html` di browser pilihan Anda.
